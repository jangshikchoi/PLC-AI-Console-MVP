# PLC RAG MVP Demo

Next.js App Router + TypeScript + Tailwind 기반의 클릭 가능한 PLC AI Console MVP 데모입니다.

## 실행

```bash
npm install
npm run dev
```

## 주요 페이지

- /dashboard
- /chat
- /admin/knowledge-base
- /admin/rules
- /admin/io-mapping

## 시나리오 데이터

- 펌프 기동 실패
- 비상 정지 회로 동작
- 탱크 고수위 알람
- 컨베이어 인터록 미충족
